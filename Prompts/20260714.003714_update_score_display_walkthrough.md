# Walkthrough: Updating Game Status Display

This document details the modifications made to improve the readability and completeness of the game's top status display.

## Actions Taken
1. **Formatting Live Score:** Edited `onScoreUpdated` in `GameActivity.java`. It now concatenates the string with a newline character (`\n`) so the coin count (`Black: X | White: Y`) sits exactly on the first line, and the player turn (`Black's turn` or `White's turn`) drops precisely to the second line.
2. **Improving Game Over Status:** Edited `onGameOver` in `GameActivity.java`. Re-factored it to retrieve both final coin counts from the `GameManager` state. It now determines the winner (including Draws) and formats a multi-line message: `Game Over! <Winner>\nFinal Score - Black: X | White: Y`.
3. **UI Alignment:** Added `android:gravity="center"` to the `scoreText` TextView in `app/src/main/res/layout/activity_game.xml`. Since it's now dynamically displaying multi-line text, this ensures both lines remain perfectly centered above the board.
