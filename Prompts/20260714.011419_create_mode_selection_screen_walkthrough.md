# Walkthrough: Dedicated Game Mode Screen

This document tracks the architectural update that replaces the basic system popup with a dedicated, themed Android Activity for selecting the game mode.

## Actions Taken
1. **Designed a New Layout:** Created a new XML layout file (`activity_mode_selection.xml`) featuring a prominent "Select Game Mode" header and two large buttons for "Human vs Human" and "Human vs Computer". It respects the app's Edge-to-Edge display paddings just like the main menu.
2. **Built the Activity Controller:** Created `ModeSelectionActivity.java`. This new screen handles the user's choice, modifies the `aiEnabled` property in the `SettingsManager`, writes that state directly to the JSON file, and then seamlessly boots up the game board while closing itself.
3. **Wired the Application:** 
   - Stripped the old unstyled `AlertDialog` popup logic out of `MainActivity.java` and pointed the "New Game" button directly to the new `ModeSelectionActivity`.
   - Formally registered the new Activity inside `AndroidManifest.xml` so the Android OS allows it to run.

The game now features a clean, professional flow from the Main Menu straight into a beautifully themed mode selection screen!
