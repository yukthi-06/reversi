# Walkthrough: Game Mode Selection Prompt

This document summarizes the changes made to the "New Game" flow to give the user immediate choice over the match type.

## Actions Taken
1. **Intercepted Action:** Instead of instantly launching `GameActivity` when the "New Game" button is tapped on the main menu, I modified `app/src/main/java/com/vypeensoft/reversi/activity/MainActivity.java` to pause the flow and display an interactive prompt.
2. **Built Selection Dialog:** Using the standard Android `AlertDialog`, the app now asks the user to "Select Game Mode" and provides a list with "Human vs Human" and "Human vs Computer".
3. **Wired State Logic:** 
   - When the user taps a choice, the app immediately updates the `aiEnabled` flag inside the `SettingsManager` and saves it to disk. 
   - Only after this preference is secured does the app construct an `Intent` and launch the game board, ensuring that `GameManager` boots up with the correct configuration!
