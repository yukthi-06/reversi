# Walkthrough: Implementing Coin Style Toggle

This document details how the setting to toggle between 3D Spherical Coins and Flat Coins was built and integrated.

## Actions Taken
1. **Model & Storage:** Added a boolean field `use3dCoins` to `SettingsManager`. Since `SettingsManager` is serialized using the `Gson` library, this immediately handles saving and loading the preference to/from the requested `/sdcard/Vypeensoft/Reversi/settings/settings.json` file.
2. **User Interface:**
   - Modified `activity_settings.xml` to include a switch labeled **Use 3D Spherical Coins**.
   - Created `SettingsActivity.java` to act as the screen controller. It reads the current boolean state, updates it when the switch is toggled, and calls `SettingsManager.getInstance().save()` when the **Save and Exit** button is pressed.
   - Wired the previously inactive **Settings** button on the Main Menu (`MainActivity.java`) to launch this new settings screen.
3. **Rendering:** Modified the `drawCoin` method inside `BoardView.java`. It now explicitly checks `SettingsManager.getInstance().use3dCoins` before drawing. If toggled on, it renders the sphere with radial gradients. If off, it renders the flat classic discs.

## Validation
You can now tap the Settings button on the main menu, flip the switch, save, and instantly see your preference applied to the game board!
