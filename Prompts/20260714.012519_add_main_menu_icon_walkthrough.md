# User Prompt
in the main screen above the word "REVERSI" place the app icon in a decent size

# Walkthrough: Adding Main Menu Icon

This document details the layout update to incorporate the app icon directly into the main menu screen.

## Actions Taken
1. **Located Main Layout:** Opened the layout file responsible for the main menu screen (`activity_main.xml`).
2. **Injected Image View:** Above the "REVERSI" text, I injected an `ImageView` and wired its source to `@mipmap/ic_launcher`, which is the official launcher icon of the app.
3. **Sizing and Spacing:** I gave the icon a decent, prominent size of `128dp x 128dp` so it stands out but doesn't overwhelm the screen. I also added a `16dp` margin below it so that the icon and the title text are neatly separated.

The main screen now proudly displays the app's Reversi coin icon right above the title!
