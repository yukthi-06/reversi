# User Prompt
make buttons 'undo' and 'restart' less width.  They are touching

# Walkthrough: Fixing Button Spacing

This document summarizes the changes made to prevent the 'Undo' and 'Restart' buttons from visually overlapping or touching in the UI.

## Actions Taken
1. **Identified Issue:** The `btnUndo` and `btnRestart` buttons inside `activity_game.xml` were using `layout_weight="1"` without any margins, causing them to span exactly half the screen each and butt up against one another in the center.
2. **Applied Margins:** Added `android:layout_marginEnd="8dp"` to the Undo button and `android:layout_marginStart="8dp"` to the Restart button.
3. **Result:** The buttons maintain their equal widths (`0dp` with weight `1`), but now have an 8dp visual gap between them, ensuring they no longer touch.
