# Walkthrough: Pretty Printing JSON Output

This document summarizes the change to ensure that human-readable configuration files are generated by the app.

## Actions Taken
1. **Identified Serialization Logic:** Found the `save()` method in `app/src/main/java/com/vypeensoft/reversi/storage/SettingsManager.java`.
2. **Enabled Pretty Printing:** Instead of using the default `new Gson()` instance (which outputs the entire JSON tree into a single, minified line), I swapped it out for a `GsonBuilder` configured with `.setPrettyPrinting().create()`.

## Result
Now, whenever you click "Save and Close" in the app, the output file at `/sdcard/Vypeensoft/Reversi/settings/settings.json` will be formatted with proper line breaks and indentation, making it much easier to read and manually edit if necessary!
